.. _bugs:

Bugs and errors
===============

Please report bugs or other problems to the developers so they can be fixed.
We need that feedback from the
community to maintain the quality of the code.

* If you are unsure if it is a real bug or a usage problem,
  consider writing to the ``ase-users``
  mailing list, forum, or chat (see :ref:`contact`).

  Please provide the failing script as well as the information about your
  environment such as versions of relevant libraries or codes and any
  output files, as well as the Python stacktrace if applicable.
  If you script is complicated, please trim it to the simplest possible
  script that can reproduce the problem.

* If you think it is a bug, you can also report it directly on our
  `issue tracker`_.  The advantage of reporting bugs
  here is that it is not forgotten (which may be a risk on the mailing
  list).

We do not guarantee to fix all bugs, but we will do our best.

.. _issue tracker: https://gitlab.com/ase/ase/issues
